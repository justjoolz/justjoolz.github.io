# JUSTJOOLZ

A living personal archive for Julian “Joolz” Rutherford: music, story, process and systems.

## Work locally

```sh
npm install
npm run dev
```

## Add music

Edit `public/content/music.json`. The catalogue, filters and media links are generated from that file.

The implementation replaced on 10 July 2026 is preserved under `archive/2026-07-10-before-brief-rebuild/`. An earlier experiment remains under `v1/`.
